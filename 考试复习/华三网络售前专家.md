# 1、路由器、交换机及其操作系统介绍

##  1、路由的作用

+ 连接具有不同介质的链路
+ 连接网络或子网，隔离广播
+ 对数据报文执行寻路和转发
+ 交换和维护路由信息



##  2、Comware的作用

+ Comware是设备运行的网络操作系统，H3C产品的
  核心软件平台

+ 对硬件驱动和底层操作系统进行屏蔽与封装
+ 集成了丰富的链路层协议、以太网交换、P路由及转
  发、安全等功能模块
+ 制定了软硬件接口标准规范，对第三方厂商提供开放
  平台与接口
